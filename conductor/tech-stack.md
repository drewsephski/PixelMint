# Technology Stack

## Core Framework
- **Frontend/Backend:** Next.js (App Router)
- **Language:** TypeScript

## Infrastructure & Services
- **Authentication:** Clerk Auth
- **Database:** Supabase (PostgreSQL)
- **Storage:** Supabase Storage (For storing generated user images and assets)

## AI & Media Engine
- **Image Generation API:** Fal.ai (Chosen for extreme speed and low-latency generation)

## UI & Styling
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide React

## Development Tools & Libraries
- **Schema Validation:** Zod
- **Form Management:** React Hook Form
- **State Management:** TanStack Query (React Query) for efficient API data fetching and caching
